# Матчинг отелей

Клиент для API матчинга отелей. Отправляет CSV-файл с внешними данными об отелях на эндпоинт `/match` и сохраняет результат сопоставления.

## Использование

```bash
python test.py
```

Скрипт отправляет файл `01_hotels_external.csv` на `http://127.0.0.1:8000/match` и сохраняет ответ в `output/matches.csv`.

## Формат входных данных

`01_hotels_external.csv` — CSV с колонками:

- `external_id` — идентификатор отеля во внешней системе
- `name` — название отеля
- `address` — адрес
- `lat` — широта
- `lon` — долгота

## Формат результата

`output/matches.csv` — CSV с колонками:

- `external_id` — идентификатор из входного файла
- `hotel_id` — идентификатор matching-отеля в целевой системе
- `score` — уверенность совпадения (0–1)
